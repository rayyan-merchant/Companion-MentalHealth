# Companion

A non-clinical wellness reflection app for Pakistani university students that combines deterministic signal extraction, rule-based reasoning, safety interception, and optional LLM-assisted phrasing.

## Features
- Conversational chat interface with support for crisis detection
- Explainable analysis of user input
- Session history management
- Optional LLM integration for natural phrasing
- Rule-based reasoning system with configurable rules

---
**Note**: Companion is not a substitute for professional mental health care. In case of emergency, please contact local crisis services.
